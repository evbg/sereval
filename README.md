## Getting Started

### Installation

#### Manual install
```
git clone https://github.com/evbg/sereval.git
cd sereval
python setup.py install
```

#### Installing directly from the [repository](https://github.com/evbg/sereval) on GitHub.com
```
pip install git+https://github.com/evbg/sereval.git
```
